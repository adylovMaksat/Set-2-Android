Description

This Android app simulates a Lamp with a Bulb:

The Bulb can be turned on/off and can burn out if overused.
The Lamp can turn on/off, brighten or dim (0–10), and only shines if the bulb works.
The app includes tests to show all behaviors: turning on/off, adjusting brightness, burning out the bulb, and replacing it.
